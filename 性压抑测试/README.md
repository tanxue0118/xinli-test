# 性压抑测试

一个在线性压抑测评网站，通过25道专业测评题目，帮助用户了解自己的性压抑程度，探索对性的态度和感受。

## 功能特点

- **五维度分析**：羞耻、焦虑、回避、态度、表达
- **三种压抑程度**：低度压抑、中度压抑、高度压抑
- **雷达图展示**：直观呈现各维度得分分布
- **详细报告**：包含个人特点和成长建议
- **响应式设计**：适配桌面端和移动端
- **纯前端实现**：无需后端服务，开箱即用

## 技术栈

- HTML5
- CSS3（渐变、动画、毛玻璃效果）
- JavaScript（原生，无框架依赖）
- Canvas API（雷达图绘制）

## 快速开始

### 本地使用

直接在浏览器中打开 `index.html` 即可使用。

### 部署到GitHub Pages

1. Fork 或克隆本仓库
2. 进入仓库 Settings > Pages
3. Source 选择 `main` 分支
4. 点击 Save，等待部署完成
5. 访问 `https://你的用户名.github.io/性压抑测试/`

### 部署到其他静态托管服务

将所有文件上传到任意静态网站托管服务即可：
- Vercel
- Netlify
- Cloudflare Pages
- 阿里云OSS
- 腾讯云COS

## 压抑程度说明

### 低度压抑
- 总体得分：低于40%
- 特点：对性持开放、健康的态度，没有过度的羞耻感或焦虑

### 中度压抑
- 总体得分：40%-65%
- 特点：对性有一定的保留和谨慎，能接受性是生活的一部分

### 高度压抑
- 总体得分：高于65%
- 特点：对性有较强的压抑和羞耻感，可能影响亲密关系

## 测评维度

| 维度 | 说明 |
|------|------|
| 羞耻维度 | 反映对性的羞耻和罪恶感 |
| 焦虑维度 | 反映对性相关事物的焦虑程度 |
| 回避维度 | 反映回避性话题和性行为的程度 |
| 态度维度 | 反映对性的整体态度 |
| 表达维度 | 反映表达性需求和欲望的能力 |
