# 依恋类型测试

一个基于依恋理论的在线心理测评网站，通过35道专业测评题目，帮助用户了解自己的依恋类型和情感关系模式。

## 功能特点

- **五维度分析**：焦虑、回避、依赖、信任、沟通
- **四种依恋类型**：安全型、焦虑型、回避型、恐惧型
- **雷达图展示**：直观呈现各维度得分分布
- **详细报告**：包含个人特点和成长建议
- **响应式设计**：适配桌面端和移动端
- **纯前端实现**：无需后端服务，开箱即用

## 技术栈

- HTML5
- CSS3（渐变、动画、毛玻璃效果）
- JavaScript（原生，无框架依赖）
- Canvas API（雷达图绘制）

## 快速开始

### 本地使用

直接在浏览器中打开 `index.html` 即可使用。

### 部署到GitHub Pages

1. Fork 或克隆本仓库
2. 进入仓库 Settings > Pages
3. Source 选择 `main` 分支
4. 点击 Save，等待部署完成
5. 访问 `https://你的用户名.github.io/依恋类型测试/`

### 部署到其他静态托管服务

将所有文件上传到任意静态网站托管服务即可：
- Vercel
- Netlify
- Cloudflare Pages
- 阿里云OSS
- 腾讯云COS

## 依恋类型说明

### 安全型依恋
- 焦虑维度：低
- 回避维度：低
- 特点：自在地建立亲密关系，信任伴侣

### 焦虑型依恋
- 焦虑维度：高
- 回避维度：低
- 特点：渴望亲密，担心被抛弃

### 回避型依恋
- 焦虑维度：低
- 回避维度：高
- 特点：重视独立，保持情感距离

### 恐惧型依恋
- 焦虑维度：高
- 回避维度：高
- 特点：既渴望亲密又害怕受伤

## 测评维度

| 维度 | 说明 |
|------|------|
| 焦虑维度 | 反映对被抛弃的恐惧程度 |
| 回避维度 | 反映对亲密关系的回避程度 |
| 依赖维度 | 反映对依赖他人的态度 |
| 信任维度 | 反映对伴侣的信任程度 |
| 沟通维度 | 反映情感表达能力 |
